# ITHub - IT Operations Command Center

ITHub is a comprehensive IT operations management platform designed to streamline department workflows, track assets, manage team members, and monitor real-time facility status.

## 🚀 Key Features

### 📊 Centralized Dashboard
* **Real-time Stats:** Instant overview of active projects, pending tasks, IT requests, and open issues.
* **Live Class Status:** Monitor room occupancy and ongoing sessions across the campus.
* **Activity Widgets:** Quick access to the most urgent tasks, projects, and system alerts.

### 👥 People & Team Management
* **People Directory:** A complete organization directory showing every member and their **assigned equipment**.
* **IT Members:** Dedicated management for the core IT team with detailed professional profiles.
* **Detailed Modals:** Deep-dive into any person's profile to see their contact info, assigned hardware, security access levels, and recent activity logs.

### 💻 Asset & Inventory Tracking
* **Hardware & Software Inventory:** Track laptops, monitors, routers, and software licenses.
* **Asset History (Records):** A full "paper trail" for every asset, showing purchase history, assignments, maintenance logs, and transfers.
* **Serial Number Tracking:** Precise identification for all physical and digital assets.

### 🎫 IT Request System
* **Request Management:** Streamlined workflow for hardware, software, and access requests.
* **Detailed Review:** Expand any request to see full descriptions, priority levels, and requester details before approving or denying.

### 🏗️ Project & Task Management
* **Project Tracking:** Monitor long-term IT initiatives with visual progress bars and checklists.
* **Task Board:** Manage daily operations with priority-based tasking and due date tracking.

### 🏢 Facility Management (Rooms)
* **Room Status:** Real-time monitoring of conference rooms and labs.
* **Capacity Tracking:** View seating capacity and current occupancy status.

### ⚠️ Issue & Alert System
* **System Issues:** Track and resolve technical problems based on severity (Low to Critical).
* **Notification Center:** Real-time alerts for system updates, task assignments, and urgent requests.

### 👤 User Profile
* **Personalized Hub:** A dedicated space for the logged-in user to manage their own profile, track their led projects, and view their security status.

## 🛠️ Tech Stack
* **Frontend:** React with TypeScript
* **Styling:** Tailwind CSS
* **Animations:** Framer Motion (motion/react)
* **Icons:** Lucide React
* **Routing:** React Router

## 📂 Project Structure
* `/src/pages`: Individual modules for each major feature.
* `/src/components`: Reusable UI components.
* `/src/mockData.ts`: Centralized data store for the application.
* `/src/types.ts`: Global TypeScript definitions.
